# site-details.json - מדריך מילוי

זהו **המקום היחיד** באתר שבו מופיעים פרטים אמיתיים של המשרד. כל רכיב באתר (כותרת, פוטר, טפסים, כפתור וואטסאפ, עמוד נגישות) קורא אך ורק מקובץ זה. שדה שערכו מתחיל ב-`PLACEHOLDER` נחשב "לא מולא" - הרכיב התלוי בו יוסתר או יציג את הממלא עד למילוי.

| שדה | מה למלא | דוגמה |
|---|---|---|
| `phone` | מספר הטלפון כפי שיוצג באתר (משמש גם לחיוג בלחיצה) | `054-1234567` |
| `email` | כתובת דוא"ל ליצירת קשר (מוצגת באתר) | `matan@abirlev.com` |
| `leadDestinationEmail` | הכתובת שאליה נשלחים הלידים. חייבת להיות זהה לכתובת שאיתה נפתח חשבון Web3Forms | `Matan@abirlev.com` |
| `officeAddress` | כתובת המשרד (מוצגת בפוטר ובעמוד צור קשר) | `דרך מנחם בגין 1, תל אביב` |
| `whatsappNumber` | מספר וואטסאפ בפורמט בינלאומי, ספרות בלבד, ללא + | `972541234567` |
| `whatsappGreeting` | הודעת פתיחה שתופיע אוטומטית בצ'אט (בעברית; מקודדת אוטומטית) | כבר מולא - ניתן לשנות |
| `officeHours` | שעות פעילות | `ימים א׳-ה׳, בתיאום מראש` |
| `linkedinUrl` / `facebookUrl` / `instagramUrl` | קישורים לרשתות (אופציונלי - ריק = האייקון לא יוצג) | `https://linkedin.com/in/...` |
| `domain` | כתובת האתר המלאה כולל https - משמשת ל-SEO, sitemap וקישורים קנוניים | `https://www.abirlev.com` |
| `accessibilityCoordinatorName` | שם רכז/ת הנגישות (מוצג בהצהרת הנגישות) | `עו"ד מתן אביר לב` |
| `accessibilityCoordinatorContact` | דרך התקשרות לרכז הנגישות (דוא"ל או טלפון) | `matan@abirlev.com` |
| `logoPath` | נתיב קובץ הלוגו בתוך `public/` | `/logo.png` |
| `portraitPath` | נתיב תמונת הפורטרט בתוך `public/` | `/images/matan.webp` |
| `googleAnalyticsId` | מזהה GA4 (אופציונלי - ריק = לא נטען כלל) | `G-XXXXXXX` |
| `metaPixelId` | מזהה Meta Pixel (אופציונלי - ריק = לא נטען כלל) | `1234567890` |

לאחר כל שינוי בקובץ יש לבנות את האתר מחדש (`npm run build`) ולפרוס.
